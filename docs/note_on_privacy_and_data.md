# Privacy and Data Availability

This project is based on handwritten prescription understanding research. Due to privacy and ethical concerns, identifiable prescription images, patient names, doctor seals, phone numbers, hospital IDs, addresses, and other sensitive healthcare information are not publicly shared.

Only anonymized diagrams, cropped non-identifiable examples, result tables, workflow illustrations, and recreated demonstration samples are included for academic presentation.

Any prescription image used in this repository should be cropped, blurred, anonymized, or recreated before public upload.
